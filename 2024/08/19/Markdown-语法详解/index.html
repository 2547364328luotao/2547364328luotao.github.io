Markdown 语法详解

强调

  • 使用星号 (*) 或下划线 (_) 来强调文本。
    • *斜体* 或 _斜体_
    • **粗体** 或 __粗体__
    • ***粗体斜体*** 或 ___粗体斜体___

示例:

渲染结果:

斜体 斜体
粗体 粗体
粗体斜体 粗体斜体

列表

  • 无序列表: 使用 -, *, + 符号。
  • 有序列表: 使用数字后跟点号 .。

示例:

渲染结果:

  • Item 1
  • Item 2
    • Sub-item 1
    • Sub-item 2
  1. First item
  2. Second item
    1. Sub-item 1
    2. Sub-item 2

链接

  • 创建链接: [链接文本](链接地址)
  • 创建图片: ![替代文本](图片链接)

示例:

渲染结果:

GitHub

代码块

  • 行内代码: 使用单个反引号 ` 包围。
  • 多行代码块: 使用三个反引号 ``` 开始和结束。

  • 表格由管道符 | 和短横线 - 构成。

Row 2, Col 1 Row 2, Col 2 Row 2, Col 3

水平线

脚注

  • Markdown 支持脚注功能，但需要解析器支持。

示例:

渲染结果:

这是一个带有脚注的句子1.

1. 这是脚注的内容. ↩

任务列表

  • 在列表项前添加 - [ ] 或 - [x] 来创建可勾选的任务列表。

示例:

渲染结果:

  • [x] 完成任务 A
  • [ ] 完成任务 B
  • [ ] 完成任务 C

附加语法

除了上述基本语法外，一些 Markdown 解析器还支持额外的功能，如表格对齐、定义列表等。

表格对齐

  • 在分割行中使用冒号 : 来设置列的对齐方式。

渲染结果:
